# git push

현재 브랜치가 추적하고 있는 리모트 브랜치에  
현재 브랜치 push

## 지정
ex)
```
push origin master
```
현재 브랜치 상관 없이  
master 브랜치가 추적중인 origin 저장소의 브랜치에  
master 브랜치 push


### push로 리모트 저장소에 브랜치 생성
```
push origin newbranch
```
